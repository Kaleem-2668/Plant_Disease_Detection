# Grape Leaf Disease Detection

An AI-powered computer vision project that automatically detects and classifies grape leaf diseases using deep learning techniques. The project demonstrates how machine learning can assist in early disease identification to support precision agriculture and improve crop management.

## 🚀 Features

* Automatic grape leaf disease classification
* Image preprocessing and data preparation
* Deep learning model training and evaluation
* Prediction on new leaf images
* Jupyter Notebook for experimentation
* Modular Python implementation
